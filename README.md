# eVerse.Server  

This repository contains the backend API for the eVerse project.  

## Description  

- Built with .NET 9.  
- Provides RESTful API endpoints.  
- Supports OpenAPI (Swagger) for API documentation.  

## Setup  

1. Clone the repository:  
   ```bash
   git clone <url>

2. Restore dependencies:
    ```bash
    dotnet restore


3. Run the server:
    ```bash
    dotnet run

4. Access the API documentation:  
Open your browser at: https://localhost:8080/index.html
